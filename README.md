# Flood Detection using Satellite Images

## Overview
This project uses deep learning to detect flood areas from satellite imagery using segmentation models and transformer-based architectures.

## Model
- Backbone: Prithvi-EO-2.0
- Framework: PyTorch

## Project Structure
- src/: Training and inference code
- notebooks/: Experiment notebooks
- models/: Best model checkpoint

## How to Run

### Install
pip install -r requirements.txt

### Train
python src/train.py

### Inference
python src/inference.py
